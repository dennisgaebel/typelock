# TypeLock.js

Clamping typography for your responsive needs using a combination of JavaScript and CSS variables.

## Usage

```html
<script src="path/to/directory/typelock.js"></script>
<script>
  addTypeLock({
	  cssvar: '--fontsize-global',
	  min: 14,
	  max: 18,
	  factor: 10
  });
</script>
```
